Tutorial

  • Phylogenetics A short walkthrough
    By clicking on the Phylogenetics button in the navigation bar, you will be sent over to another page. You can fill in the form with an EGGnog ID, which can be found under the tab Result Table.
  • Sunburst A short walkthrough
    By clicking on the Sunburst button in the navigation bar, you will be sent over to another page. You are able to click on the circle on all the different layers. By clicking layer by layer you can select Stresses, Organisms, Orthologs and genes.
  • Result Table A short walkthrough
    By clicking on the Result Table button, you can insert a keyword, which generates a table with all kinds of information about the hit.
  • Force Layout Graphs A short walkthrough
    By clicking on the Force Layout Graphs button, you will be redirected to a page with a graph. By clicking on nodes, you will be redirected to a more informative table. Basic relations are shown in the graph.
  • Programming information A short walkthrough
    By clicking on the Programming information tab you can select all kinds of different tabs which contain more information about the programming side of the project.
